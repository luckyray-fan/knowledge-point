## 如何提高首屏加载速度

- js 文件放 body 底部, css 放 head
- http 静态资源用多个子域名
- 服务端开启 gzip, 响应添加 expires
- 减少 http 请求数量
- 图片用雪碧图, 大图用懒加载
- 使用异步脚本, 动态创建脚本

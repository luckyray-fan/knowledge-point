## GraphQL 和 RESTful 的比较

作为两种 API 模型, GraphQL 是为了改进 Rest 而出现的

- rest, 核心思想是资源, 每个资源能用一个 URL 来表示
- GraphQL, 资源的描述信息与其获取方式相分离
